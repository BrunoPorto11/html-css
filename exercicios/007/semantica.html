Exemplo de semantica:

Forma:

Rua alfazema, 2 - jardim flor de maio

para que tenha um destaque neste endereço, foi usado a tag <u>
que na realidade é uma tag de forma, ou seja, para estilização, o qual o CSS3 é responsavel e ja não corresponde ao HTML5

significado:

Rua alfazema, 2 - jardim flor de maio

para que tenha um destaque neste endereço, foi usado a tag <address>
que na realidade é uma tag com significado, pois é propia para endereços, e se o arquivo html for aberto em um smartphone, esse destaque encaminhará para o GoogleMaps
